What is Deck

From digital files to cassette-ready WAV

Deck is a desktop app built for people who still love the ritual of recording to tape. Pick your formulation and tape length, drop in a track or a whole folder, and let Deck handle loudness, EQ, and headroom so your music survives the transfer to analog.

Features

Everything you need before you hit record

Drop & prepare

Start fast — drag a file onto the window or import a folder for a full mixtape.

  • Drag-and-drop audio files or folders
  • One-click Prepare with adaptive mastering
  • Before / after preview with waveform comparison

Tape setup

Match your deck and blank tape. Deck adapts processing to the profile you choose.

  • Type I (Normal), Type II (Chrome), Type IV (Metal)
  • C60, C90, C120 or custom minutes per side
  • Quick presets — Normal, Chrome, and Metal deck

Mixtape builder

Planning a full cassette side? Import a folder and arrange tracks before mastering.

  • Folder import with automatic track ordering
  • Track list editor — reorder and trim the lineup
  • Fit check — see if your selection fits the chosen tape length

Export ready

When it sounds right, export a clean WAV and head to your deck.

  • True-peak limiting and cassette-aware EQ
  • Optional preflight tones for level calibration
  • Export WAV — ready for line-in or headphone jack recording

FAQ

Common questions

Do I need a DAW or plugin host?

No. Deck is a standalone desktop application. Import audio, prepare it, and export WAV — all in one window.

Which audio formats can I import?

WAV, FLAC, AIFF, and OGG. Export is WAV.

macOS says the app is from an unidentified developer

Release builds are not notarized yet. Right-click Deck.app → Open, then confirm. You only need to do this once.

How is this different from normal mastering?

Deck targets cassette transfer: tape formulation profiles, HF headroom, true-peak limiting, and optional tape-aware saturation — so your digital master survives recording to analog and playback on a deck or walkman.

Can I build a full mixtape from a folder?

Yes. Import a folder, arrange tracks in the editor, check whether everything fits your tape length, then prepare and export.

How do I build from source?

See the README on GitHub. You need CMake 3.22+ and Xcode Command Line Tools (macOS) or Visual Studio 2022 (Windows).
